# The Geography of Despair
### Drug Overdose Mortality × Economic Distress — United States, 1999–2022

A professional Streamlit dashboard built on two federal datasets. 
No fabricated data. Complete data pipeline included.

---

## The Story

Drug overdose is now the leading cause of accidental death in America — 
surpassing car accidents, firearms, and falls. But it is not uniformly distributed.
The overdose crisis has a geography that maps almost exactly onto economic abandonment:
the counties and states that lost manufacturing and mining jobs in the 1990s–2000s
are the same communities with the highest overdose mortality rates today.

**Four research questions, answered in sequence:**
1. Where is the crisis worst? (choropleth map)
2. When did each wave arrive? (annotated timeline)
3. Does poverty predict overdose? (scatter with trendline)
4. Which states bear the heaviest burden? (ranked bar)

## Data Sources

### Source 1 — CDC WONDER Multiple Cause of Death Database
- **What it is:** Federal mortality database — every death certificate filed in the US
- **URL:** https://wonder.cdc.gov/mcd-icd10-provisional.html
- **ICD-10 codes used:** X40–X44 (accidental poisoning), X60–X64 (intentional), X85, Y10–Y14 (undetermined)
- **Metric:** Age-adjusted death rate per 100,000

**How to download (manual — CDC has no public API for this):**
1. Go to https://wonder.cdc.gov/mcd-icd10-provisional.html
2. Section 1 → Group results by: **State**
3. Section 2 → Select years: **1999 through 2022** (check "All Dates")
4. Section 4 → ICD-10 Codes: expand "External Causes" → select X40–X44, X60–X64, X85, Y10–Y14
5. Section 6 → Check "Age-adjusted Rate" → per 100,000
6. Send → Export → Save as **cdc_overdose.txt** in this directory

If the file is absent, `pipeline.py` falls back to published CDC summary 
statistics from NCHS Data Brief No. 491 (November 2023).

### Source 2 — US Census Bureau, American Community Survey 5-Year Estimates (2022)
- **What it is:** The most comprehensive annual US demographic/economic survey
- **API endpoint:** https://api.census.gov/data/2022/acs/acs5
- **Variables pulled:**
  - `B17001_002E` — Population below poverty level
  - `B17001_001E` — Total population (poverty universe)
  - `B19013_001E` — Median household income
- **No API key required** for standard ACS queries

`pipeline.py` pulls this automatically. If the Census API is unavailable, 
it uses hardcoded 2022 ACS published estimates.

## Quick Start

### Option A — Run locally

```bash
# 1. Clone or download this folder
cd despair-dashboard

# 2. Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the data pipeline (downloads Census data, processes everything)
python pipeline.py

# 5. Launch the dashboard
streamlit run app.py

```

## Design Decisions

### Why Streamlit, not Tableau?
Tableau requires a license ($840+/yr) for full features and cannot be 
deployed as a single standalone artifact without Tableau Server or Tableau Public.
Streamlit is free, open-source, and the entire dashboard is one Python file 
(`app.py`) deployable with a single command. For a shareholder presentation,
Tableau Public is acceptable — but the dashboard cannot be self-hosted.

### Why these chart types?
- **Choropleth map:** Geographic pattern is the central argument. A map is the 
  only chart that proves "this crisis has a geography." Plotly's choropleth 
  renders state-level data with hover tooltips and accurate boundaries.
  
- **Annotated line chart:** The overdose crisis unfolded in three distinct waves 
  driven by different substances. A timeline with wave shading and event annotations 
  communicates causation, not just trend. The year-range slider lets users zoom 
  into specific waves.
  
- **Scatter with trendline:** Correlation is the analytical core of the story. 
  A scatter plot is the *only* chart type that shows a relationship between two 
  continuous variables. The trendline makes the direction of correlation 
  immediately visible. Region highlighting adds a layer of interactivity.
  
- **Ranked horizontal bar:** Simple, fast, legible. Answers "who" in under 
  3 seconds. Horizontal orientation allows full state names without rotation.
  Color intensity encodes severity without a separate legend.

### Why no pie chart, no stacked area?
Pie charts require angular comparison — humans perform this task poorly.
Stacked areas require comparing heights against a moving baseline — also 
cognitively expensive. Every chart here has one primary visual task: 
read a position, read a length, or read a color intensity. 
All three complete in under 4 seconds.

### Color system
- Dark amber/red sequential scale = severity. Color encodes one thing only: 
  how bad the overdose rate is. The same scale appears on both the map and 
  the scatter plot, so viewers learn it once.
- Teal = correlation statistic (positive signal in the data context)
- Warm amber annotations = structural labels (wave regions, events)
- Dark ink background for masthead = editorial weight, not decoration

## Extending This Dashboard

**Add county-level data:**
The CDC WONDER export supports county-level grouping. 
Download with "County" in Section 1, then use `plotly.express.choropleth_mapbox` 
with FIPS county codes and a Mapbox tiles background.

**Add a year-by-year animation:**
The `pipeline.py` script can be extended to pull state-level rates for every year.
Use `px.choropleth` with `animation_frame="year"` to create a play-button animation.

**Add a predictive model:**
With the merged dataset, a simple OLS or ridge regression 
(`sklearn.linear_model`) can estimate overdose rate from poverty, income, 
and employment variables. Display residuals to show which states outperform 
or underperform the economic prediction.

## Data Integrity Notes

All overdose rates are **age-adjusted** — meaning they account for 
differences in age distribution between states and over time. 
This is the methodologically correct comparison metric.
Raw death counts would overstate risk in older states and understate it in younger ones.

Suppressed values in CDC WONDER (states/counties with <10 deaths in a year) 
are excluded from analysis. This affects a small number of cells in early years.

Income and poverty figures are from the 2022 ACS 5-year estimates, which 
represent an average of 2018–2022 — not a single-year snapshot.
